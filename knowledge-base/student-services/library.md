# UPOU Library Resources and Services

The UPOU Library is a core component for scholarly research, providing information services in various formats to students and faculty.

## Library Services Table
| Service | Description |
| :--- | :--- |
| **Web OPAC** | Online Public Access Catalog via the UP System Integrated Library Management System. |
| **Virtual Reference** | Request books, chapters, or full-text articles via email. |
| **Document Delivery** | Sending and receiving printed books via courier service. |
| **Bibliographical** | Preparation of subject and author bibliographies upon request. |
| **Referral & Loans** | Intra/Inter-library loans for materials outside the UP System. |
| **Reader’s Assistance** | Library orientation, query answering, and research material advice. |

# Course Development: Resource-Based Course Package (RBCP)

Since 2007, UPOU has transitioned to a fully online course delivery model. The university uses the **Resource-Based Course Package (RBCP)** approach, which is faster and more cost-effective than traditional print-based materials.

## Components of an RBCP
A complete course package includes:
1. Study Guide and Course Guide
2. Online learning resources and multimedia materials
3. Discussion forum questions
4. Assignment guides and formative assessment tools
5. Final examination or terminal requirements

## Development Workflow
1. **Basis:** Guided by the approved course syllabus.
2. **Management:** Program Chair coordinates with **OASIS** and the **Multimedia Center (MC)**.
3. **Execution:** Involves writing workshops, multimedia production, and material review.
4. **Implementation:** Faculties-In-Charge populate the **MyPortal** course site before the term begins.
